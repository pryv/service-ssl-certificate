# service-ssl-certificate

Pryv.io service for SSL certificate generation and renewal

Prerequisites: Node v16+, Yarn v1+

## How to?

| Task                              | Command                        |
| --------------------------------- | ------------------------------ |
| Setup                             | `yarn`                         |
| Run                               | `yarn start`                   |
| Run Tests                         | `yarn test`                    |
| Run Tests with logs               | `yarn test-logs`               |
